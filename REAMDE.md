# README FOR git-demo

Working with git

- Workflow of git:
   1. `git init` once to initialize the repo
   2. Write some code
   3. `git add .` / `git commit -m "Some message` whenever you want to make a commit
   4. `git push origin main` push the changes to github
   5. repeat steps 2-4

- Create branches with `git switch -c`
